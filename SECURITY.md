# Reporting A Security Issue

Please do not publish passwords, tokens, setup codes, private URLs, or a
working exploit in a public issue.

If this repository has GitHub private vulnerability reporting enabled, use the
**Security** tab to submit a private report. Otherwise, contact Autark Labs at
<licensing@autarklabs.com> with **Security report: Autark-OS** in the subject.

Include the affected Autark-OS version, operating system, a safe reproduction,
and the impact you observed. Do not attach unredacted support bundles or
credentials. Autark Labs will acknowledge the report, assess the impact, and
coordinate a fix or public advisory when appropriate.

This is a community product support path, not a promise of a particular
response time or security certification.
