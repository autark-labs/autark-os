# Autark-OS Product Terminology

Use these terms in user-facing copy, activity messages, installer output, and documentation.

| Concept | Canonical wording | Rule |
| --- | --- | --- |
| App catalog | Discover | Do not call the user-facing destination Marketplace. |
| Current-instance apps | My Apps / managed apps | Call an app installed only when this instance owns it. |
| Host-detected resource | Found on this server | Include ownership context when known. |
| External shortcut | Linked service | It is visible and openable, not managed. |
| Lifecycle | Pause / Resume | Use Stop only in a final shutdown or removal plan. |
| Backup state | Protected by a restore point | Only after a completed, verified restore point. |
| Support export | Support report | Reserve bundle for an archive or CLI artifact. |

Technical names such as Docker Compose, catalog IDs, runtime paths, and API routes belong in Diagnostics or advanced disclosure.
